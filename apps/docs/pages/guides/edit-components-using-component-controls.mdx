---
title: See Your Component In Different States Using Component Controls
description:
  With Triplex your component props become controls you can modify during
  development enabling to easily see your component over different states
  without writing extra code.
---

Component controls can be used to change the currently opened components props
that persist only for the duration of your session. Click the "Open Component
Controls" action in the scene panel to have the component controls be displayed
in the context panel.

![Triplex UI](/ui/live-edit-props.png)

After you're done you can click the "Reset Props" action in the context panel to
reset the prop values back to default.
