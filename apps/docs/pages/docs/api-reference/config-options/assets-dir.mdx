---
app: Standalone
title: assetsDir
description: Name of the folder that will have its contents appear in the Assets Panel.
---

Defaults to `"assets"`.

## Usage

Declare the name of the folder inside the [publicDir](./public-dir) that will have its contents appear in the Assets Panel.

```json filename=".triplex/config.json"
{
  "assetsDir": "my-assets"
}
```
