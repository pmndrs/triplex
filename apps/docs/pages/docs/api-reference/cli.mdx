# CLI

The Triplex CLI allows you to start your app, follow
[setup Node.js](/docs/setup/nodejs) to make it available.

The get a list of available commands run the following command inside your
project directory:

```bash
npx triplex -h
```

The output should look like:

```bash
Usage: triplex [options] [command]

The React Three Fiber editor.

Options:
  -V, --version     output the version number
  -h, --help        display help for command

Commands:
  editor [options]  start the editor
  help [command]    display help for command
```

## Editor

Start the Triplex editor by running:

```sh
npx triplex editor
```

### Options

| Option                    | Default   | Description                                           |
| ------------------------- | --------- | ----------------------------------------------------- |
| `-E --export-name <name>` | `default` | Export of the file to use when opening a file.        |
| `-o --open [path]`        |           | Opens your default browser with an optional filepath. |
