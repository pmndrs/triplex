---
description: Logs are used to investigate issues with your scene and Triplex.
---

## Triplex Standalone

When running Triplex Standalone you can access the logs by clicking in the menu View > Logs.

## Triplex for VS Code

When running Triplex for VS Code you can access the logs by accessing the command palette via <Kbd>Shift</Kbd> + <Kbd>Cmd</Kbd> + <Kbd>P</Kbd> and typing "Output: Show Output Channels..." then selecting "Triplex".
