Transform Controls appear on selected 3D elements, used for moving, rotating, and scaling scene objects.

Transform Controls are enabled only for elements that declare the prop for the given mode. Otherwise it is disabled.

- **position** — Enables translate mode for the scene object
- **rotation** — Enables rotate mode for the scene object
- **scale** — Enables scale mode for the scene object

## Transform Modes

The [Scene Controls](/docs) contains the transform mode actions, interacting with them sets the mode accordingly. For example clicking the <ActionLink icon={AngleIcon} href="/docs/building-your-scene/learn-the-ui/scene#rotate" name="Rotate" /> action enables rotate mode.

Each mode has a corresponding hotkey you can use instead of clicking the action.

- Translate — <Kbd>T</Kbd>
- Rotate — <Kbd>R</Kbd>
- Scale — <Kbd>S</Kbd>

## Turn Off Transform Controls

Turn off Transform Controls clicking the <ActionLink icon={CursorArrowIcon} href="/docs/building-your-scene/learn-the-ui/scene#select" name="Select" /> action in the Scene Controls.
