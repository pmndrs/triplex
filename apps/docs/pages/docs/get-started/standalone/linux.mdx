## Installation

1. Download [Triplex Standalone for Linux](/download)
1. Right-click the AppImage and allow it to execute as a program
1. Move the AppImage to a folder in your `PATH` to make it easier to launch [optional]
1. Open Triplex by double clicking the AppImage

## Common Questions

### Something isn't working

Linux support for Triplex is currently experimental. Please give try it out and give feedback / raise issues on [GitHub](https://github.com/trytriplex/triplex) or [Discord](https://discord.gg/nBzRBUEs4b).
